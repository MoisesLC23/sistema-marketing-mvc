# Sistema de Optimización de Marketing

Sistema web desarrollado para la gestión y optimización de estrategias de marketing en el sector de telecomunicaciones, estructurado bajo el patrón MVC para asegurar un código limpio y escalable.

## Tecnologías Utilizadas
* **Backend:** PHP
* **Base de Datos:** MySQL
* **Frontend:** HTML, CSS, JavaScript

## Instalación y Configuración
1. Clonar este repositorio.
2. Importar el archivo `database.sql` (o la estructura de la base de datos) en tu gestor MySQL.
3. Renombrar el archivo `app/config/config.example.php` a `app/config/config.php` y colocar tus credenciales locales.
4. Ejecutar `composer install` para descargar las dependencias.
